```mermaid
sequenceDiagram
    autonumber
    participant UI as <<boundary>><br/>Create_Character_UI
    participant Battle as 전투
    participant P as 플레이어(Player)
    participant Char as 캐릭터(Character)

    UI->>Battle: 캐릭터 생성 요청(id, 이름, 직업, 레벨)
    activate Battle
    
    Battle->>P: 플레이어체크(id)
    activate P
    P-->>Battle: 체크 결과 (boolean)
    deactivate P

    alt id == "hero"
        alt 직업 == "전사"
            Battle->>Char: 전사 객체 생성(이름, 레벨)
            activate Char
            Char->>Char: this.HP = 레벨 * 100
            Char->>Char: this.공격력 = 레벨 * 15
            Char-->>Battle: 전사 객체 반환
            deactivate Char
        else 직업 == "마법사"
            Battle->>Char: 마법사 객체 생성(이름, 레벨)
            activate Char
            Char->>Char: this.HP = 레벨 * 60
            Char->>Char: this.공격력 = 레벨 * 25
            Char-->>Battle: 마법사 객체 반환
            deactivate Char
        end
        Battle-->>UI: 캐릭터 생성 완료 반환
    else id != "hero"
        Battle-->>UI: 생성 실패 (ID 불일치)
    end
    
    deactivate Battle
